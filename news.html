Version 0.2.0

Released on jun 11, 2016

  1. New option for converting html code into code for embedding on your website - Convert the characters '&', '<', '>', " (double quote), and ' (apostrophe), in a string to their corresponding HTML entities.
  2. Changing parts of the page inside nested divs is now possible
  3. Templates are imported when application starts.
  4. New icon
  5. Added .desktop file that runs the program in one click
    Please make sure, you change the Icon path and Path in .desktop file in order to run Html Assistant properly. If you are planning to run in via cmd line, you don't have to change anything.
